User Host=(User:Group) Command

User - пользователь к которому применятся правило. Если начинается с процента (%) то это группа пользователей.

Host - на каких хостах будет применяться правило.

(User:Group) - пользователь и группа из под которых команда может может быть запущена

Command - какие команды можно выполнять 

cloud	ALL=(ALL:ALL) ALL - по сути полный доступ
