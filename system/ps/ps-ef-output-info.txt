UID – user id - пользователь
PID – process id – идентификатор процесса
PPID – parent process id – идентификатор родительского процесса
C – использование процессора данным процессом
TTY - с каким терминалом ассоциируется данный процесс - https://habr.com/ru/articles/460257/
TIME – сколько времени процессор потратил на работу с данным процессом
CMD – команда, которая запустила процесс
